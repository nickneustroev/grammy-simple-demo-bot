# Grammy Simple Bot

Простой Telegram бот на grammy, который отвечает "привет" на любые сообщения.

## Установка

1. Склонируйте репозиторий или перейдите в папку проекта

2. Установите зависимости:
```bash
npm install
```

## Получение токена бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

## Настройка

1. Создайте файл `.env` в корне проекта
2. Добавьте ваш токен бота:
```
BOT_TOKEN=ваш_токен_здесь
```

## Запуск

Запустите бота:
```bash
npm start
```

Для разработки с автоматическим перезапуском:
```bash
npm run dev
```

## Использование

Отправьте любое сообщение вашему боту в Telegram, и он ответит "привет".
